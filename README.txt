----------------------- To run the server: ------------------------------

1. Join the root folder of the project trough your terminal
2. command "make all" - generates all executables
3. command "make install" - copies the exectuables to the ./bin folder
4. enter the folder 'bin' trough the terminal, use several terminals. 
   One for each server and atleast one for a client.
5.  "cache_server localhost 1337"	- Creates a server with cache database
	"persistant_server localhost 1338" - Creates a server with cache database
	"client 1337"	- Client connects to server with port 1337
	"client 1338"	- Client connects to server with port 1337

--------------------------------------------------------------------------
#commands:
			make all	- generates all necessary files
			make clean	- cleans produces files from "make all"
			make install - copies all exectuables (if produced) to ./bin
			make uninstall	- Deletes all exectuables in the .bin folder
